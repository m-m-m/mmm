Use mmm
Write one client, get all for free!
mmm-client
  • API for all types of widgets
  • Implementation for different native UI toolkits
  • JUnit testing of UI including interaction
  • High-level patterns and features
  • Usability & accessibility
  • Dialog composition framework with history management
  • Good performance via lazy widget creation
mmm-service
  • API for asynchronous RPC communication
  • Implementation for GWT-RPC and Spring-Remoting
  • Batching calls in single HTTP request
  • Service interface, implementation with @Named - done
  • Dive into service impl from client call with one click in IDE
mmm-persistence
  • API and base implementation for persistence layer
  • Based on JPA and best practices
  • Generic and consistent access to entities and DAOs
  • Support for custom data-types
  • Typesafe builder for dynamic JPQL queries
mmm-util
Generic library for
  • Native-language-support
  • Command-line-interfaces
  • Value conversion
  • Reflection (generic types, etc.)
  • POJO introspection and navigation
  • Text hyphenation and line-wrapping
  • Unicode support (e.g. resolve combining chars)
  • handling of dates, streams, processes
  • etc.
mmm
  • Apache Software Lincense 2.0
  • Excellent JavaDocs
  • Modular and extendable
  • CDI based, best with spring
  • Available in central maven repository
  • Get it now at m-m-m.sf.net
